# Deany - Learn Islam Beautifully

Interactive Islamic education app with gamified lessons covering:
- **Islamic Finance** - Principles, riba, gharar, maysir, salam, istisna, governance
- **Islamic History** - From pre-Islamic Arabia through the Prophet's mission
- **5 Pillars of Islam** - Coming soon
- **Quran & Arabic** - Coming soon

## Setup

```bash
npm install
npm run dev
```

## Deployment

Connect this repo to [Vercel](https://vercel.com) for automatic deployment.

## Adding Content

Edit the module data inside `src/App.jsx`. Each module follows this structure:
- **Concept cards** - Educational content shown before questions
- **Questions** - Multiple types: MCQ, fill-in, drag-sort, compare, calculate, advise
- **Speed Round** - Rapid triage classification
- **Master Quiz** - Cross-lesson synthesis assessment
